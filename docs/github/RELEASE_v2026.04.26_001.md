# SnapSlate v2026.04.26_001

SnapSlate v2026.04.26_001 sharpens the product around the core flow: capture, annotate, and export.

## Highlights

- Procedure view centered on the canvas, with retractable steps and a compact tool surface.
- Real Windows capture import still creates one step per screenshot while the app is open.
- Contextual properties for selected annotations, including opacity for every item type.
- Cleaner Capture/Help, Export, and Settings screens.
- Real app screenshots and a multilingual public site for the Archi-IT Labs suite.
- A bilingual installer in French and English.

## Included in this release

- Automatic import of `Win + Shift + S` captures while SnapSlate is open.
- One tab per capture.
- Crop, text, sticker, straight arrow, curved arrow, rectangle, oval, spotlight, masking, delete, duplicate, undo, and redo.
- Ten gradient palettes and seven shades per palette.
- Opacity controls for all objects.
- Sticker renumbering when deleting an item or switching color, with the option to restart numbering.
- PNG, PDF, DOCX, Markdown, and HTML export.
- A clearer, more production-oriented interface.

## Release asset

- `SnapSlate-Setup.exe`

## Notes

- The clipboard-based capture flow still requires SnapSlate to be open.
- The website opens in the browser language when supported and defaults to English.
- The website language can still be changed manually between English and French.
