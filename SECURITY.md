# Security Policy

## Versions supportees

| Version | Support |
| --- | --- |
| 1.x | Oui |

## Signaler une faille

Merci de ne pas publier une faille de securite sensible dans une issue publique.

Merci de preferer:

- un contact prive avec le proprietaire du depot
- ou une GitHub Security Advisory si le depot l'active

Quand tu remontes un probleme, essaye d'inclure:

- le contexte
- la version de `SnapSlate`
- les etapes de reproduction
- l'impact estime
- une preuve de concept si elle est utile et sure

Une reponse initiale est generalement attendue sous quelques jours ouvrables.
